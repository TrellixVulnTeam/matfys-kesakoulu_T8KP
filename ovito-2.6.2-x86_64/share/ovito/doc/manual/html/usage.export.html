Exporting data

To export particles to a file (or a series of files), select File → Export File from the menu. After choosing an output filename and a file format, the export settings dialog lets you specify the animation frames and the particle properties to be exported.

If you pick an output filename that ends with the .gz suffix, the file will be automatically compressed by OVITO.

Note that OVITO exports the output of the modification pipeline. That means, for instance, if you have deleted some particles using a modifier, OVITO will export only the remaining particles to the output file. The modification pipeline is re-evaluated for every animation frame being exported.

Supported output formats

File formatDescription
LAMMPS dump

Text-based file format produced and read by the LAMMPS molecular dynamics code.

LAMMPS data

File format read by the LAMMPS molecular dynamics code.

XYZ

A simple column-based text format, which is documented here.

POSCAR

File format used by the ab initio simulation package VASP.

IMD

File format used by the molecular dynamics code IMD.

FHI-aims

File format used by the ab initio simulation package FHI-aims.

Crystal Analysis (CA)

A text-based file format used by the Crystal Analysis Tool, which can store dislocation line networks extracted by the Dislocation Analysis modifier.
