Histogram

This modifier computes and displays a histogram of a particle property.

Optionally, the modifier can be used to select particles whose property value is within a certain interval.

Parameters

Property

The particle property the histogram should be computed for.

Number of histogram bins

Controls the resolution of the generated histogram.

Select particles in range

If this option is active, the modifier selects all particles whose property value falls into the specified range.
